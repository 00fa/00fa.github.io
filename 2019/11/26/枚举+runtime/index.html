枚举+Runtime

发表于 2019-11-26 | 分类于 Java ， 基础

枚举的手动设计，属性，常用方法，Runtime三个常用的管理内存的方法

枚举类

若没有枚举类型，可以手动设计，将每个对象一一列举，

private 的构造方法( )(为了不让别人调用)

public static final Day monday = new Day（）（所有对象都是属性，public让别人可以用，final又让其不能被修改,static是为了保证只创建一个）

特性：

  • enum类型默认继承自Enum（java.lang包），所以不能extends其他父类
  • 有两个属性：
    • name——枚举对象的名字 ，name() 可以获取name属性
    • ordinal—–枚举对象的顺序，类似于index ，ordinal() 获取序号
  • 常用的方法：
    • valueOf() ———–通过给的name获取对应的枚举对象
    • values()————–获取全部的枚举对象，通过数组返回
    • compareTo()——–比较两个枚举对象
    • toString()———— 这个可以重写
  • 注意的事项:
    • 自己定义构造方法必须是private修饰（也可以不写，默认也是这个）
    • 在第一行描述枚举的对象，最后以分号结束。

Runtime

提供了几个管理内存的办法

  • maxMemory

  • totalMemory

  • freeMemory

  • 堆内存溢出错误OutOfMemoryError

  • 栈内存溢出错误StackOverflowError
